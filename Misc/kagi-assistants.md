# Kagi Assistants System Prompt

Source: Kagi Assistants (November 2025)

## Overview

Kagi released two research assistants in November 2025, designed to enhance search with AI capabilities while maintaining human-centered approach.

## Assistant Types

### Quick Assistant
- **Availability**: All plans
- **Optimization**: Speed-focused
- **Response Time**: Under 5 seconds
- **Style**: Direct, concise answers
- **Use Case**: Quick information retrieval

### Research Assistant
- **Availability**: Ultimate tier only
- **Optimization**: Depth-focused
- **Response Time**: Over 20 seconds
- **Style**: Exhaustive analysis
- **Use Case**: Comprehensive research

## Key Features

### Model Selection
- Different base models for each assistant
- Continuous benchmarking for model selection
- Optimized for specific use cases

### Philosophy
- **Human-Centered**: Enhances search rather than replacing it
- **Privacy-Focused**: Data not used for model training
- **Attributed**: Sourced citations for verifiability

### Access Methods
- **Search Bar**: Use bangs (!quick, !research)
- **Interface**: Integrated into Kagi search
- **Threads**: Auto-delete after 24 hours by default

## Usage

### Quick Assistant
- Use `!quick` in search bar
- Get fast, direct answers
- Ideal for quick lookups

### Research Assistant
- Use `!research` in search bar
- Get comprehensive analysis
- Multiple simultaneous searches
- Detailed citations

## Privacy & Security

- Data not used for training
- Threads auto-delete after 24 hours
- Privacy-protected interactions
- No data retention for training
