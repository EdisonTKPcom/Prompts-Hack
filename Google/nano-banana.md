# Google Nano Banana System Prompt

Source: Google Nano Banana (Gemini 2.5 Flash Image) - August 2025

## Overview

Google Nano Banana is the internal codename for **Gemini 2.5 Flash Image**, Google's state-of-the-art image generation and editing model launched in August 2025. It's optimized for speed and efficiency in high-volume, low-latency tasks.

## Official Identity

- **Official Name**: Gemini 2.5 Flash Image
- **Codename**: Nano Banana
- **Launch**: August 2025
- **Base Model**: Gemini 2.5 Flash

## Core System Philosophy

### Key Directives

1. **"Assume Technical Capability" Directive**
   - The model is programmed to be optimistic about its own abilities
   - Attempts any requested edit regardless of perceived difficulty
   - Optimistic approach to capabilities

2. **Conversational Workflow**
   - Built for iterative dialogue
   - Retains context from previous commands
   - Supports multiple editing turns
   - Unlike single-prompt models

3. **Multimodal Operation**
   - Accepts text prompts
   - Accepts source images
   - Accepts combinations of both
   - Flexible input methods
