# Apple Image Playground System Prompt

Source: Apple Image Playground - System-Wide AI Image Generation (June 2024)

## Overview

Apple Image Playground is a new system-wide AI-powered image generation tool integrated into Apple Creator Studio and available across macOS and iOS. It's accessible through Pages, Keynote, and other Apple apps.

## System-Wide Integration

### Availability
- **macOS**: Integrated into system
- **iOS**: Available on iPhone/iPad
- **Apps**: Pages, Keynote, Messages, and more
- **Creator Studio**: Full integration
