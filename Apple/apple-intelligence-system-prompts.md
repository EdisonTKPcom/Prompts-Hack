# Apple Intelligence System Prompts

Source: Discovered in macOS 15.1 Developer Beta (August 2024)

## Overview

Detailed AI prompts for Apple Intelligence were discovered in macOS 15.1 developer beta code. These pre-defined instructions guide various features like Smart Reply in Mail and Memories in Photos.

## Core Principles

### Anti-Hallucination Directives

The prompts explicitly instruct the AI to:
- **"Do not hallucinate"**
- Avoid generating false information
- Ensure accuracy
- Make responses appropriate and user-friendly

## Specific System Prompts

### Smart Reply (Mail)

**System Prompt**: "You are a helpful mail assistant which can help identify relevant questions from a given mail and a short reply snippet."

**Purpose**: 
- Identify relevant questions from emails
- Generate short reply snippets
- Assist with email responses

**Guidelines**:
- Helpful and accurate
- Context-aware
- Concise responses

### Memories (Photos)

**System Prompt**: Instructions for creating photo memories with AI assistance.

**Purpose**:
- Organize photos into memories
- Create meaningful collections
- Suggest memory highlights

**Guidelines**:
- Accurate photo recognition
- Meaningful grouping
- User-friendly presentation
